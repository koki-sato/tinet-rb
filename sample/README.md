# Sample

Samples of tinet configs. You can test them in [Vagrant](https://www.vagrantup.com/).

## Requirements

- Vagrant

## Setup

```bash
$ vagrant up
$ vagrant ssh

# in vagrant shell
$ cd tinet/sample
$ bundle install
```

## Usage

```bash
# in vagrant shell
$ bundle exec tinet [OPTIONS] COMMAND
```

## Discard

```bash
$ vagrant destroy
```
